# Apparatus Layout Morph

A premium scroll-driven component where layout is interpolated dynamically based on the scroll position. It transitions seamlessly through intermediate states (Grid → Masonry → Horizontal Strip).

## Features
- Dynamic layout keyframes calculated on mount and resize.
- Smooth GSAP-driven scroll scrubbing.
- Interactive controls for card count, sequence ordering, travel rotation, and easing curves.
- Custom momentum tilt that rotates cards towards their movement vector.
