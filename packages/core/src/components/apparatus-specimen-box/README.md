# Apparatus Specimen Box

Images sit as pinned entomological specimens in a velvet case that tilts with cursor movement; clicking an image slides out the metal pin, lifting the specimen to float at center.

## Usage

```tsx
import { ApparatusSpecimenBox } from "@vessel/core";

<ApparatusSpecimenBox
  imageSrc="/showcase/1.jpg"
  images={[
    "/showcase/1.jpg",
    "/showcase/2.jpg",
    "/showcase/3.jpg",
  ]}
/>
```
