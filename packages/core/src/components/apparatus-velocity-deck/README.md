# Apparatus Velocity Deck

Scroll-pinned image deck that physically stretches, skews, and splits into offset slices in response to scroll velocity.

## Usage

```tsx
import { ApparatusVelocityDeck } from "@vessel-ui/core";

<ApparatusVelocityDeck
  imageSrc="/images/landscape.jpg"
  scrollProgress={0.5}
/>
```
