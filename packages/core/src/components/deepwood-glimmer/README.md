# Deepwood Glimmer - Velvet Tapestry

A static image representation of a heavy medieval velvet tapestry forest.

## Usage

```tsx
import { DeepwoodGlimmer } from "@vessel-ui/core";

function App() {
  return (
    <DeepwoodGlimmer 
      imageSrc="/path/to/forest.jpg"
      className="w-full max-w-md"
    />
  );
}
```
