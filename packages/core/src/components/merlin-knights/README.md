# Merlin Knights - Voronoi Glass Shatter

An interactive Voronoi glass shatter effect. Clicking anywhere shatters the glass panel at the impact point, scattering refracted shards in 3D space under gravity before they pull back and weld together.

## Usage

```tsx
import { MerlinKnights } from "@vessel-ui/core";

function App() {
  return (
    <MerlinKnights 
      imageSrc="/path/to/image.jpg"
      className="w-full max-w-md h-[400px]"
    />
  );
}
```
