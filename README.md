# AbsoluteUI

A curated collection of motion-crafted React components designed to create immersive digital experiences.

AbsoluteUI is not a utility-first library. It is a beauty-first, curated design system focusing on visual polish, exceptional motion, and production-ready implementation.

## Tech Stack

* **Core:** Next.js, React, TypeScript, Tailwind CSS
* **Animation:** GSAP (GreenSock), Framer Motion, Lenis Scroll, Motion One, Rive
* **Visuals:** WebGL, OGL, React Three Fiber (R3F)
