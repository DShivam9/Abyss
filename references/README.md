# Design References & Inspiration

Welcome to the Reference Folder. This directory is reserved for dumping design systems, guidelines, and `DESIGN.md` files of other websites and libraries (e.g., Magic UI, Shadcn, Aceternity, Apple) to serve as a high-fidelity reference.

## Workflow

1. **Dump Reference Files**: Create a folder for any site you like (e.g., `references/linear/`) and place their design specifications, component guidelines, or markdown files inside it.
2. **AI Reference Search**: The AI agent will inspect this folder for visual design guidelines, animation easing curves, and layout principles to ensure all newly constructed or refactored components meet or exceed these industry standards.
3. **Reference Guidelines**:
   * Inspect color tokens (HSL tailor systems, monochrome structures).
   * Study easing curves and animation parameters (GSAP durations, spring dynamics).
   * Analyze spatial layout proportions and grid rhythms.
