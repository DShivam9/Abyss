# Component Ideas

This document stores creative proposals, interaction styles, and experimental ideas for future AbsoluteUI components.

---

# Brainstorming Pool

## 1. Liquid Canvas Button
* **Interaction**: Mouse hover distorts a fluid SVG filter layer around the button boundaries.
* **Tech**: GSAP + Custom SVG filters.

## 2. Parallax Magnetic Card
* **Interaction**: Hovering scales down background images while tilting layers in 3D perspective space, pulling target buttons magnetically.
* **Tech**: Framer Motion 3D transforms.

## 3. Kinetic Text Wipe
* **Interaction**: Scroll-bound masking reveals text lines with a smooth spring-based character-stagger rotation.
* **Tech**: GSAP ScrollTrigger + SplitText.
